This is a project developed and maintained by the UPU to ensure that partner DOs are compliant with all of the EDI (Electronic Data Interchange) requirements. This report is linked to multiple reports which shares KPIs from this project and their respective project. Such as:

  • EMS Performance Awards
  • IQRS Data Reports
  • EAD (Electronic Advanced Data) Compliance
  • ICS2 EU Regulatory requirements
  • Customs Compliance

Project and Report Scope

The project is cross-cutting and covers all mail products, regardless of the EDI network used for message exchanges and the mode of international mail transport. Compliance checks are performed for all EDI messages and for the most widely used technical standards, such as those governing international mail processing centres (IMPCs) and identifiers.

This report checks certain KPIs against set standards:

  • EDI messages compliance:

    • common EDI issues
    • EMSEVT, ITMATT / ITMATT received / ITMREF
    • PREDES / RESDES, PRECON / RESCON, CARDIT, CARDIT EAD

  • Compliance, other aspects:

    • own IMPCs / partner IMPCs
    • EDI connectivity, message upgrades
    • Flight quality
    • Dispatch duplicates
    • EMSEVT repeated events, EMA-EMC mismatch
    • Postal addresses in ITMATT
